# Balls of Steel Docs

This folder tracks the current product truth for Balls of Steel as of April 8, 2026.

The app is now positioned as a direct-distributed macOS trading desk for discretionary SPY and VXX options work. It is manual-first, local-first, and explicit about the difference between setup detection and actual option execution quality.

## Current Product Principles

- Local-first
- Mac-first
- Manual-first
- Broker-agnostic
- Underlying-led setup detection
- Human-confirmed options execution
- SPY and VXX only until the edge is clean enough to expand

## Notes On Scope

The VXX-specific documents in this folder still matter because they reflect a real edge and a real workflow. They are no longer the entire identity of the app.
